# Acopio — Distribution Order System (demo)

Static prototype of a 10-center distribution ordering flow:
order builder → 30% online deposit → confirmation → staff pickup & balance collection.

Pages: `index.html` (order), `deposit.html` (deposit payment), `pickup.html` (staff).
Demo only — payments are mocked, orders persist in browser localStorage.
To be rebuilt as a full React app with a real backend.
